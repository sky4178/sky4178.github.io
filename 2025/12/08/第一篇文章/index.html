博客启航

欢迎来到我的个人博客！这是我用 Hexo 和 GitHub Pages 搭建的第一个站点，记录学习与成长的点滴。

为什么选择 Hexo + GitHub

  • Hexo：轻量、快速，支持 Markdown，适合写作和技术分享。
  • GitHub Pages：免费托管，全球访问速度快，适合个人博客。

搭建过程简述

  1. 安装 Hexo 和相关依赖
  2. 初始化博客项目
  3. 选择并美化主题
  4. 部署到 GitHub Pages

后续计划

  • 分享技术文章与学习笔记
  • 记录生活与成长
  • 持续优化博客内容与样式

感谢你的访问，欢迎留言交流！
